PDML Companion Installation

To install the PDML Companion application (PDMLC), proceed as follows:

  • If not done already, install Java version 17 or later.

    Ensure that your OS PATH environment variable includes the bin sub-directory of your Java installation.

  • Extract one of the following files into a directory of your choice:

    • macOS/Unix/Linux: pdmlc-java-app-0.79.0.tar (run tar -xf pdmlc-0.79.0.tar)
    • Windows: pdmlc-java-app-0.79.0.zip

  • Add the path to the bin sub-directory of the PDML Companion to your OS PATH environment variable.

  • To get the list of all available CLI commands, open a terminal and type:

    pdmlc help
